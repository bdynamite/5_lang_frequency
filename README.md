# Frequency Analysis of Words

Вставьте текстовый файл в папку со скриптом и запустите его.
Скрипт выведет 10 самых популярных слов в порядке уменьшения.

1) ("мама", 56)
2) ("папа", 37)
3) ("деньги", 30)
...
